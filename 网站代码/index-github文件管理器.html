使用说明 ▼

1. 在"访问令牌"输入框中输入你的GitHub Personal Access Token

2. 点击"加载文件"按钮查看文件列表

3. 不同文件类型有不同的操作：

  • 📄 文本文件：可编辑
  • 🎵 音乐文件：可在网站内播放
  • 🖼️ 图片文件：可在网站内查看
  • 📁 文件夹：可进入并上传文件
  • 📎 所有文件：都有下载功能

4. 新建文件夹功能错误，请点击新建文件功能，格式是***/**.keep

5. 要上传文件到文件夹，请先进入文件夹，然后使用上传区域
